Slot Content

슬롯(slot)은 컴포넌트의 재사용성을 높여주는 기능이다. 특정 컴포넌트에 등록된 하위 컴포넌트의 마크업을 확장하거나 재정의 할 수 있다.

<navigation-link> 컴포넌트 태그 안에 각기 다른 내용을 정의했다. 만약 NavigationLink 컴포넌트에 slot을 정의하지 않았다면 컴포넌트를 등록하는 시점에 마크업을 재정의 할 수 없었을 것이다.

이처럼 슬롯을 사용하면 컴포넌트의 특정 마크업 영역을 재정의 해 같은컴포넌트르 각기 다르게 표현할 수 있다.

Slots 은 HTML을 포함한 템플릿 코드 또는 컴포넌트로도 재정의 할 수 있다.

Named Slots Shorthand

New in 2.6.0+

v-slot은 #으로 대체해 사용 가능하다. (예) v-slot:header => #heaer

그러나 다른 지시어와 마찬가지로 축약어는 argument가 제공된 경우에만 사용할 수 있으므로 아래 코드는 유효하지 않다.

대신 축약어를 사용하려면 특정한 slot 이름을 지정해야한다.

Javascript 배열 메소드들

2019-03-26

참고

https://youtaekjung.github.io/2018/04/30/js/JS-Array-method/

Array.from()

인자가 배열인지 확인하는 메소드

Array.from(arrayList[, mapFn[, thisArg]])

return Array

Array.isArray

value가 배열인지 체크하는 메소드

Array.isArray(value)

return boolean

Array.of()

숫자 인자로 부터 새로운 배열을 만드는 메소드

Array.of(element0[, element1[, ... [, elementN]]])

return Array

Array.prototype.concat();

두 개 이상의 배열을 합쳐주는 메소드

var new_array = old_array.concat([value1[, value2[, ...[, valueN]]]])

return Array

Array.prototype.copyWithin()

사이즈는 변하지 않지만 배열의 순서를 변경해서 옅은 복사를 하는 메소드

arr.copyWithin(target[, start[, end]])

return modified Array

처음으로 함수 실행 시간을 체크해 볼 일이 있었다.

window의 perfomance 객체를 활용하는 방법이다.

이렇게 사용하면 된다~!

이것에 찾아보다가 아직 사용해본 적이 없는 console.time에 대해서도 알게 되었는데

time() 메서드는 새로운 타이머를 시작하고, 소요된 시간을측정하는데 매우 유용하다. 문자열의 해당 메서드에 전달해 마커의 이름을 지정하고 타이머를 중지하려면 timeEnd()를 호출하고 initializer에 전달한 것과 동일한 문자열을 전달하면 콘솔이 timeEnd() 메서드가 실행될 때 경과한 시간과 레이블을 기록한다.

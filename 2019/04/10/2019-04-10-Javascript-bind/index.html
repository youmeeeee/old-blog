Javascript 함수 기본 메소드 call, apply, bind

2019-04-10

참고

https://www.zerocho.com/category/JavaScript/post/57433645a48729787807c3fd

나도 경험이 좀 더 쌓이다 보면 이런것들이 전부 익숙해질까.. 헷갈리는 것들이 많은 js…

먼저 call과 apply 메소드는 함수를 호출하는 방법이다.

call

call은 보통 함수와 동일하게 인자를 넘기고, apply는 배열로 묶어서 넘긴다.
여기서 중요한 것은 공통으로 넘긴 null 인자인데… 이것은 Javascript의 this를 대체한다.

내 경우obj의 check 메소드를 호출할 때 check 메소드 내부의 this가 obj2가 된다…라고 이해된다. (즉 다른 객체의 함수를 자기 것 마냥 사용할 수 있다.)

call, apply, bind 메소드에서 첫번째 인자로 다른 것을 넣어주는게 this를 바꾸는 방법 중 하나이다.

이제 bind 함수에 대해서 알아보자. bind 함수는 함수가 가리키는 this만 바꾸고 호출하지 않는 것이다.

call과 apply처럼 this를 바꾸지만 호출을 하지 않고 함수만 반환한다…

Why are my contributions not showing up on my profile?

2018-04-04

바보 인증 주의!!!

열심히 커밋 하는데 프로필에 커밋 로그(잔디밭..이)가 안뜬다…

왜안되지.. 왜 안되지..하고 그냥 방치한 어리석은 지난 날은 지금 여기서 반성…합니다.

셋팅도 public only에서 private 커밋도 포함하게 바꿨는데도 안뜬다. 하아…

Why are my contributions not showing up on my profile? 를 다시 한번 봤다. 아무리 봐도 해당사항이 없는데 왜안뜨지…

이메일을 한번 더 확인해 보자.. (몇번이고 봤던것 같지만)

git config --list 명령으로 user.email을 확인했다.

이메일이 youmechoi1129@gmail.com 로 되어있는 걸 확인..

분명 github 셋팅도 해당 이메일로 해둔걸로 알고 있었는데..

학!

youmekko1129@gmail.com 로 되어있다…

바보인증

이메일 주소가 비슷해서.. 셋팅이 잘 되어있다고 생각했었나 보다.

email 셋팅을 youmechoi1129@gmail.com 로 바꾸려고 시도 해보았다.

…

이미 사용중이란다.

(사용하지 않는 다른 아이디가 하나 더 있다.. )

git config --global user.email 이메일주소 명령으로 로컬 이메일 셋팅을 다시 했다.

변경이 잘 되었나 확인해보니 잘 되어있다!

이젠 잘 뜨겠지…?

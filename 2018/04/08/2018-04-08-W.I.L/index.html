느낀 점

  • 왜 안되지.. 하고 콘솔에 로그 찍다보면.. 변수명 오타로 값이 안 넘어 와서일때가 많다. 주의하자.

  • JSP 게시판의 주요 기능(읽기 / 수정 / 삭제)이 다 되기도 하고, 그냥 ‘’다음에 오라클에서 시퀀스 써보자’’ 하며… 어제까지 한걸로 마무리 할까 싶었는데 그래도 마음먹고 다 고쳐놓고 나니까 기분이가 좋다.

  • Ajax 강좌 짧아서 금방 보려니 했는데 중간부터 아예 왜 안되는지 모르겠다. 서버 접속한 후 어떤 액션을 해도 콘솔에 아무것도 안찍힌다.. Ajax 어렵다. 계속해서 트라이하는 수 밖에…

    • 중간부터 안되서 ‘자야지’ 하다가 결국 끝까지 다 해보긴 했다. 스크립트랑 제이쿼리에 대한 이해도 약하고 Ajax의 흐름도 헷갈린다. 어디서부터 어디까지 스크립트 함수가 꼭 필요한건지 모르겠다.. 일단 완성 시켰으니까 코드 보고 또 봐야지.

  • 부스트코스 첫번째 프로젝트는 html / css 기본 활용을 위한 프로젝트다. 어쩐지 약한 부분이다.. 시간이 많이 필요한 프로젝트는 아니지만, 이번 주 내 언제할 수 있을지 아직 모르겠다… 이번주 완성하기를 목표로 해야겠다.

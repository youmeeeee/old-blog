2018-04-16

생명 주기(life cycle)와 범위(scope)

DI를 이용해서 Ioc컨테이너를 만들 수 있다.

  • 스프링 컨테이너 생명 주기
  • 스프링 빈 생명주기(1)

ctx.close() 의 경우 컨테이너가 소멸하는 단계이다. 컨테이너가 소멸하면 빈은 자동 소멸된다. 빈만 소멸하게 하려면 destroy API를 이용하면 된다.

  • 스프링 빈 생명주기(2)
  • 스프링 빈 scope

스프링 컨테이너가 생성되고 스프링 빈이 생성 될 때, 생성된 스프링 빈은 scope를 가지고 있다. 범위란 해당하는 객체가 어디까지 영향을 미칠지 결정하는 것이라고 생각하면 된다.

외부 파일을 이용한 설정

  • Enviroment 객체를 이용한 설정

  • 프로퍼티 파일을 이용한 설정

    • 스프링 설정 XML 파일에 프로퍼티 파일을 명시한다.

    • 스프링 설정 JAVA 파일에 프로퍼티 파일을 명시한다.

  • 프로파일(profile) 속성을 이용한 설정

동일한 스프링 빈을 여러개 만들어 놓고 상황(환경)에 따라 적절한 스프링 빈을 사용할 수 있다.

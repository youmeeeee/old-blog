(웹동네) Typescript

2018-08-23

Typescript 핵심 내용

Typescript는 MS에서 개발한 만든 오픈 소스다. C#, Pascal, 델파이, Visual Basic 만든 엔더스 헬스버그 아저씨가 만들었다. 대규모 JS프로젝트를 만들기 위해서 만들어 졌다.

JS와 관계

  • JS 문법을 그대로 사용한다. (ES6)

  • 타입 제공(정적 타입 언어)

  • generic / enum / 모듈 기능 제공

  • 객체 지향 프로그래밍 문법 제공 (핵심)

    • 클래스 (public / protected / private)
    • 상속 / abstract / interface

일단 typescript로 개발한 소스를 자바스크립트 소스로 컴파일하고(웹팩 등 사용), 만들어진 자바스크립트 소스를 일반 js 파일 사용하듯 html 파일에 연결해서 사용한다.

준비 : typescript 컴파일러, node.js (typescript 컴파일러는 node.js기반하에 만들어졌다. )

자바스크립트 함수

(에 대해 새로 알게 된 것들!)

함수 리터럴과 익명 함수

데이터를 만드는 방법은 크게 리터럴 방식과 객체 방식 두가지가 있다.

리터럴 방식 var name="youmekko"

객체 방식 var name = new String("youmekko")

자바스크립트에서 함수를 만드는 세 가지 방법

(1) 함수 선언식 (일반적으로 가장 많이 사용한다.)

(2) 함수 표현식 (리터럴 방식) (익명함수 등에 사용)

(3) 객체 방식 (제일 사용 안함)

(4) 익명함수를 함수 확장을 이용해 정의하는 방법도 있다. 주로 모듈을 만들 때 사용한다. 함수를 만들어서 재사용 하는 것이 목적이 아니라 다른 함수간의 충돌을 막기 위해서 사용한다. 일단 있다는 것만 알아두자.

익명 함수

익명함수란 함수 리터럴 방식으로 만들어진 이름 없는 함수를 말한다. 이벤트 처리등에 익명 함수를 사용하면 더 간결해진다. 재사용이 필요 없을 때 사용한다.

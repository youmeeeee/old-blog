2018-01-09

1장. SELECT 명령을 이용하여 데이터를 조회합니다.

  1. 테이블에 어떤 컬럼이 있는지 조회하는 경우

  2. 모든 컬럼 조회하기

  3. 사용자가 만든 모든 테이블을 조회하고 싶을 때

  4. 원하는 컬럼만 조회하기

  5. SELECT 명령에 표현식을 사용하여 출력하기

  6. 컬럼 별칭 사용하여 출력하기

  7. DISTINCT 명령어로 중복된 값을 제거하고 출력하기

  8. 연결 연산자로 컬럼을 붙여서 출력하기

  9. 원하는 조건만 골라내기 WHERE절 사용하기

  10. SQL에서 기본 산술 연산자 사용하기

  11. 다양한 연산자를 활용하는 방법

  12. 정렬하여 출력하기 ORDER BY 절 사용하기

  13. 집합 연산자

    • 집합 연산자는 실제 업무에서 아주 많이 사용이 되고 있고 문제도 많이 만들고 있기 때문에 정말 조심해서 써야한다.
    • 주의사항)
      • (1) 두 집합의 SELECT 절에 오는 컬럼의 개수가 동일해야 한다.
      • (2) 두 집합의 SELECT 절에 오는 컬럼의 데이터 형이 동일해야 한다.
      • (3) 두 집합의 컬럼명은 달라도 상관 없다.
      • UNION : 두 집합의 결과를 합쳐서 출력, 중복 값 제거하고 정렬함
      • UNION ALL : 두 집합의 결과를 합쳐서 출력. 중복 값 제거 안하고 정렬 안함.
      • INTERSECT : 두 집합의 교집합 결과를 출력 및 정렬함
      • MINUS : 두 집합의 차집합 결과를 출력 및 정렬함. 쿼리의 순서가 중요함.

2018-01-10 W.I.L

2018-01-10

오늘 수업시간에는 첫번째 프로젝트에 대해서 발표하는 시간을 가졌다. 내용은 대동소이 했고 강사님의 평가는 수업과 병행하면서 진행한거라 시간이 많이 부족했음 + 첫번째 프로젝트였음 에도 불구하고 잘 해냈다고 평가해 주셨다. う。。。うれしい。개인 사정으로 이틀동안 결석하면서 팀프로젝트의 코딩에 기여한 부분이 적어졌고, 전체 코드도 아직 완벽하게 이해한 것이 아니라서 개인적으로 리팩토링을 해보고 싶은데, 대체 언제 해볼 수 있을지 기약하기가 힘들다. 현재는 매일매일 진도 따라가기도 벅차고, 그 밖에도 공부해야 할 것이 많고, 시간은 너무 제한적이다. 이 과정이 다 끝날 즈음에는…!

오늘 한 일

  • 다양한 예제로 쉽게 배우는 오라클 SQL과 PL/SQL 책 2장(SQL 단일행 함수를 배웁니다) 정규식 함수 부분 빼고 다 봤다. 정규식 함수 부분은 일단 패스한다.
  • 아침 통학 지하철에서 HavardCS50 Topic3 (지하철에서 봐서 아직 정리 못했다.)
  • 점심시간에 생활코딩 지옥에서 온 GIT 동영상 7개 정도 봤다. 시간 날때마다 틈틈히 봐야지.
  • 노트북에 Oracle XE(Express Edition) & Oracle SQL Developer 재설치 했다! 잘 된다!

오늘 느낀 점

  • SQL은 생각보다 어려운 문제가 많다. (도대체 어떻게 생각했던거지..) 2장 끝냈는데 다른 친구들은 JOIN? 부분을 배우고 있다. 나도 빨리 같이 문제 풀고 싶은데.. 수업하는 내내 혼자 책보며 공부했다. 이번주는 다른것보다 SQL 위주로 빠르게 봐야겠다. 책에 나와 있는 문제를 전부 풀어보고 넘어가고 있는데 2장(초반)인데도 막히는 부분이 많다.. 오늘 수업시간 나간 진도는 꽤 어려운지 강사님이 내주신 문제에 다들 끙끙 대었다.
  • 노트북에 Oracle XE(Express Edition) & Oracle SQL Developer 재설치 할 때 강사님이 올려주었던 설치과정 이미지를 보고 따라했다. 포인트는 오라클 접속용 방화벽을 설정하는 것인데 1521 로컬 포트를 설정해주니 문제 없이 돌아간다. 방화벽도 컴퓨터 사용하면서 자주 등장하는 단어지만, 이해는 없다. 조만간 공부하게 되겠지…!
  • HavardCS50 강의 재밌다. 전공 수업처럼 딱딱한게 아니라 일반인을 대상으로 하는 교양수업 같은 느낌. 강의별 시간이 짧아서 몰아서 보면 금방 볼것도 같은데, 시간이 없다 정말. 짬짬히 보자!
  • 지옥에서 온 GIT도 드디어 보기 시작했다. 나중에… 조만간… 해야지 하고 미루고 미뤘던 GIT 공부. 이번 프로젝트 하면서 공부의 필요성을 다시 한번 느꼈기에 더 이상 미루지 않고 시작했다. 생활코딩 동영상 역시 짧고 굵으므로 꼭 따로 시간 내려고 하지 않더라도, 점심시간이라던지 시간 있을 때 마다 틈틈히 보자.
  • 어제부터 오늘 점심까지, 낭비하는 시간 없이 최대한 공부에만 시간을 할애하려고 했는데, 고작 딱 이 하루 쉬지 않고 달렸더니 오늘 오후에는 넉다운 되버렸다. 역시.. 쉬엄쉬엄하자. 점심시간에는 좀 엎드려 있는다던지… 오늘도 일찍 자야 내일 컨디션을 유지할텐데.. SQL 진도가 뒤쳐져 있는게 신경이 쓰여서 잠을 잘 못자겠다..

내일 할 일

  • 딴 거 같이 할 여력이 없다. 오라클 진도 계속해서 따라잡자.

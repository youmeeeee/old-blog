OracleSQL(1) DB / DBMS / SQL

2018-01-07

DB

  • 데이터를 저장해 놓은 곳

DBMS (DataBase Management Software)

  • 사람이 할 수 없는, 데이터를 관리해 주는 다양한 전문적인 프로그램
    그 중에 가장 유명하고 기준이 된느 프로그램이 오라클(Oracle)이고, 오라클 이외에도 MS의 mMySQL이나 IBM의 DB2 같은 프로그램 들이 있다. 사람들이 워드 작업을 잘하기 위해서 워드 프로세서를 사용하는 것처럼 많은 데이터를 잘 관리하고 활용하기 위해서 DBMS 프로그램을 사용하는 것이다.

SQL

  • SQL이란 사람이 원하는 바를 DBMS 프로그램에 알려주는 언어이다. SQL = Structured Query Language(직역하면 구조화된 쿼리 언어 = 규칙이 있는 말)

시작 인덱스는 전체 길이에서 1을 빼고 2로 나눈 몫은 시작인덱스 (길이가 홀수이든 짝수이든 변동 없음)
종료 인덱스는 전체 길이를 2로 나눈 몫에 1을 더하면 종료 인덱스(불포함) : 길이가 홀수이면 시작 인덱스보다 1이 크고, 길이가 짝수이면 시작 인덱스보다 2가 크다.

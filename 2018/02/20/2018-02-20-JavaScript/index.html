생활코딩 WEB2 - JavaScript

  • 기본적으로 JS는 HTML 위에서 동작하는 언어이다.
  • 어떻게 하면 HTML이란 컴퓨터 언어에 완전히 다른 문법을 가지고 있는 JS를 잘 끼워넣을 것인가…
  • HTML은 정적 (예) 1+1라고 작성하면 1+1이 나온다.
  • JS는 동적 (예)1+1이라고 쓰면 2가 나온다.
  • HTML 위에서 < script > 라고 써줘야 JS라고 인식한다. 위치는 html의 body안. (혹은 style태그 아래)
  • 개발자 도구 콘솔에서 파일을 만들지 않고도 JS코드를 즉석에서 실행할 수 있다.

웹 페이지를, 또 JS이용한다 라고 하는 것은 웹페이지를 만드는 것, 또 웹 사이트를 만드는 것 처럼 거대한 목표일 수도 있지만, 이미 만들어져 있는 웹사이트를 나의 필요에 따라서 맥락적으로 , 간단하면서도 허접하고, 하지만 나에게는 딱 맞는 코드를 작성해서 그 문제를 해결 할 수 있다는 것이다. 그래서 우리가 코딩을 해서 얻을 수 있는 효용은 개발자가 될 수 있다 될수 없다만 있는 것이 아니라, 나의 현실의 문제를 해결할 수 있는 수많은 단계들의 심도들이 아날로그적으로 펼쳐져 있다.

… 여기까지는 정리하면서 듣다가 이 후로는 실습만 따라했다.

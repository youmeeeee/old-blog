(프로그래머스) Lv1. 정수 제곱근 판별하기 Python/JS

2018-02-23

문제

nextSqaure함수는 정수 n을 매개변수로 입력받습니다.
n이 임의의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 임의의 정수 x의 제곱이 아니라면 ‘no’을 리턴하는 함수를 완성하세요.
예를들어 n이 121이라면 이는 정수 11의 제곱이므로 (11+1)의 제곱인 144를 리턴하고, 3이라면 ‘no’을 리턴하면 됩니다.

[JS] 내가 푼 코드

빠르게 풀었다! 3항연산자 사용 할 일이 많다…
중요한 것은 JS의 내장객체 Math의 메소드
Math.pow(대상숫자, 거듭제곱횟수) : 대상숫자의 거듭 제곱값을 계산해준다.
Math.sqrt(대상숫자) : 대상 숫자의 제곱근의 값을 계산해준다.
제곱근의 값이 Integer인리 float인지 확인하는 방법은 나누기 1을 했을 때 나머지가 0이면 된다!
square root = 제곱근

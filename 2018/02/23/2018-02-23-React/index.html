ReactJS로 웹서비스 만들기 DAY2

Nomade Academy

  • 리액트에는 2개의 주요컨셉이 있다. 하나는 state, 나머지 하나는 props이다.

  • 부모 컴포넌트는 칠드런 컴포넌트에게 props를 통해서 정보를 준다.

  • 오브젝트(props)가 있으면, 괄호를 한번치고 this.props.”title” 이렇게 써주면 된다.

    • 이번 실습의 경우 movie > title이 있고 movie.js에서 title을 props로 불러온다. (movie 컴포넌트에 title이라는 요소가 있다.)

  • JSX 경우 명령을 시키려면 괄호를 꼭 쳐야한다.

  • 데이터의 소스는 한군데뿐이다. 메인 컴포넌트가 데이터를 다 가지고 있다.

    • 타이틀, 영화 포스터 정보르 메인에 다 집어넣고, 그걸 각각 컴포넌트에 props를 이용해 정보를 출력한다.
  • 리액트 데이터 플로우 : 큰 컴포넌트가 전체정보를 가지고 있고, 각기 다른 칠드런에 정보를 전달한다. 이를 이용해 강력한 UI를 구축 할 수 있다. (한 개의 데이터 소스를 가지고 각 컴포넌트별로 출력만 하면 된다.)

오늘은 여기까지~! (별로 못했다. 동영상 딱 1개 봤다.)

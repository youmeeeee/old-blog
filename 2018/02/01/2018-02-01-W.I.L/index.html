느낀 점

  • 알고리즘 공부 어렵게 생각하지말고 쉬운 것부터 하루 한문제씩 풀어보자는 마음으로 시작했다. 지금은 알고리즘 문제랄까.. 자바 문법(혹은 API)에 대한 테스트 같지만 그것만으로도 도움이 된다. 평소에 사용하지 않는 메소드들은 금새 잊어버리는데 따로 클래스별 메소드 정리하는 파일을 정리했다가 포스팅 해야겠다.
  • 지금 보고 있는 오라클 교재로는 정리가 잘 안되는 부분도 있고, 지금 내가 적어도 어디까지 공부해야하는지 파악이 조금 어렵다. 추천 도서에 정리해둔 SQL 책으로 mySQL 실습하면서 나중에 또 보자.
  • 두번째 프로젝트는 스키마, 화면 설계를 끝내고, 분담해서 다음 작업 진행중이다. 자세한 것은 팀 프로젝트 카테고리에 추가 포스팅 할 예정!
  • T.I.L을 주기적으로 포스팅 하기엔 내 학습 패턴이 그리 일정하지 못하다. 생활 패턴은 크게 벗어나는 부분이 없는데 매일매일의 체력 상태. .컨디션에 따라 학습량에 차이가 있다. 앞으로도 이런식으로 비정기적인 포스팅이 될 것 같다. (Today I Learned 의 의미는 무색해지지만…)
  • 한가지 일만 계속 하면 금새 지겨워짐을 느꼈다. 지금 정도로 여러가지 것을 조금씩이라도 매일 공부하자. 동영상 보며 공부한 내용은.. 따로 정리할 수 있을지 아직은 잘 모르겠다. git에 대한 정리가 필요할 것 같긴한데…

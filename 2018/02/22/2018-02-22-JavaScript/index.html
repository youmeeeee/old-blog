Do it 자바스크립트(1)

2018-02-22

DAY-1

  • var 변수선언이 필요하다. (파이썬이랑 헷갈림)

  • 객체의 종류

    1. 내장객체 : String, Date, Array, Math 등
    2. BOM (Browser Object Model) : window, screen, location, history, navigator 등(아래 그림이 이해하기 쉬워서 그려 봄)
    3. DOM (Document Object Model) : (X)HTML 문서 구조를 말한다. (X)HTML 문서의 기본 구조는 최상위 객체로 html이 있고, 그 하위 객체로는 head와 body가 있다. 자바스크립트를 이용해 src속성을 바꾸고자 한다면, 지정된 img를 선택해 src속성을 바꿔줘야 한다. 이 때 지정요소를 잘 선택해 가져오려면 문서 객체의 구조를 잘 이해하고 있어야 한다. DOM에서는 (X)HTML의 모든 요소들을 문서 객체로 선택해서 자유롭게 속성을 바꿀 수 있고, 선택한DOM에 원하는 CSS를 적용할 수 도있다.

  • 내장객체

    • 내장 객체 생성은 new 연산자를 사용한다. (자바와 동일)
    • 내장 객체 메소드는 일단 따로 정리하지 않는다. 필요할때마다 찾아쓰기.

  • BOM

    • 계층적 구조로 이루어져있다.

    • window 객체 : BOM의 최상위 객체이다.

      • open() : 새 창 열기

      • alert() : 경고창

      • prompt() : 질의응답 창

      • confirm() : 확인/취소창

      • 그 밖에 movieTo(), resizeTo(), setInterval(), clearInterval(), setTimeout(), clearTimeout() 등

    • Screen 객체

      • 사용자 모니터 정보(속성)을 제공하는 객체이다.

      • 기본형

    • Location 객체

      • 브라우저의 주소창에 ulr에 대한 정보(속성)와 새로고침 기능(메소드)을 제공하는 객체이다.

      • 기본형

      • 속성 : href, hash, hostname, host, port, protocol, search

      • 메소드 : reload() : 새로 고침이 일어난다.

    • Histroy 객체

      • 사용자가 방문한 사이트 중 이전에 방문한 사이트와 다음 방문한 사이트로 다시 돌아갈 수 있는 속성과 메소드를 제공한다.

      • 기본형

      • 속성 : length : 방문 기록에 저장된 목록의 개수

      • 메소드 : back(), forward(), go(이동숫자)

    • Navigator 객체

      • 현재 사용자가 사용하는 브라우저의 정보와 운영체제의 정보를 제공하는 객체

      • 기본형

      • 속성 :

        • addCodeName : 브라우저의 코드명 리턴

        • appName : 브라우저의 이름 리턴

        • appVersion : 브라우저의 버전정보 리턴

        • language : 브라우저의 사용언어 리턴

        • product : 브라우저 사용 엔진 이름 리턴

        • platform : 브라우저를 실행하는 운영체제 리턴

        • userAgent : 브라우저와 운영체제 종합정보 제공

오늘은 여기까지~!

(프로그래머스) Lv1. 피보나치 수 /Java

2018-02-14

문제

피보나치 수는 F(0) = 0, F(1) = 1일 때, 2 이상의 n에 대하여 F(n) = F(n-1) + F(n-2) 가 적용되는 점화식입니다. 2 이상의 n이 입력되었을 때, fibonacci 함수를 제작하여 n번째 피보나치 수를 반환해 주세요. 예를 들어 n = 3이라면 2를 반환해주면 됩니다.

문제를 보고 손으로 생각해 본 것을 이미지로 만들어 봤다.

내가 처음에 푼 코드

이렇게 풀었는데.. 프로그래머스 정답체크에서 정답으로 인정되서 제출이 되었다가.. 안되었다가 하는 오류가 있었다.. 다른 사람이 푼 코드를 보고 나니 내가 뭔가 잘못한것 같긴한데.. 리턴 값은 잘 나오는것…?
결과적으로 argument의 값이 커질 수록 리턴 값이 엉뚱해지는 상황을 발견.. 다시 풀어야지..

다른 사람이 푼 코드를 돌려 봤다(1)
40이상의 수를 넘겨줄수록 처리속도가 급격히 느려진다. 70을 넘겨주면 리턴값이 출력 되지도 않았다.

다른 사람이 푼 코드를 돌려봤다(2)

빠르고 정확하게 잘 출력된다. argument가 161일때까지(9217463444206948445) (long의 범위(9223372036854775807)까지)

여기까지 해보고 나서 내 코드를 다시 봤다. 왜 처음부터 first, second 변수 선언을 int로 하고 쓸데없는 강제 형변환을 한거지? (변명을 해보자면 그냥 int가 익숙해서…) long으로 바꿔봤다.

빠르고 정확한 값이 나온다! 헤헿. 자료형 선언에 미스가 있었던 것만 빼고는.. 빠르게 풀렸다. 기분이가 좋다!

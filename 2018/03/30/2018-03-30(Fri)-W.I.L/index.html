한 일

느낀 점

  • 오랜만에 헬로 코딩 알고리즘 책 봤다. ebook으로 구매해서 수업시간엔 보기 좋다. 알고리즘은 읽을 땐 이해가 되는 것 같다가도.. 어렵고 그렇다. 일주일에 한 챕터라도 보고 정리하자라는 목표를 세웠었는데.. 가능한 빠르게 책을 다 보고 여러 예제를 풀어보는게 더 좋을 것 같다. 기본적인 알고리즘 개념을 알고나면 내가 지금까지 풀었던 몇 안되는 기초 문제들도.. 다른 실마리가 생길 것 같다.

  • JSP 게시판 만들기 다봤다! 부트스트랩과 JSP로 최소한의 간단한 로직만 사용해 개발하는 튜토리얼 이었지만, 어쨌든 뭐 하나 내 손으로 완성시키고 나니 왕뿌듯!(학학) 시간을 따로 정해서 전반적인 코드 흐름을 몇번 더보고 open api 를 붙여보는 작업을 해보고 싶다. 다음주부터 교육센터 프로젝트 진행기간이니 시간 분배를 어떻게 할지 좀 지켜보자.

    <인프런에 남기려다 실패한 수강 소감> 처음 동영상 소개에 누구나 따라 할 수 있을거라는 멘트 때문인지 답답해하시는 댓글이 많은 것 같네요. 완강 해보니 확실히 JSP, bootstrap, html, css 에 대한 기초적인 개념이 없으신 분들은 어려울 거라는 생각이 드네요. 부분 부분 설명이 디테일하지 않은 점도 있고… 제 경우는 jsp 기초 학습 직후 였기 때문에 더할 나위 없이 좋은 강의였습니다. 기초 문법이나 개념 정리해주는 강의들만으로는 해소되지 않는 답답함이 있는데, 다 이해하지 못하더라도 이렇게 하나의 프로젝트를 진행해보는 것은 성취감도 주고 큰 틀을 보게 해주네요. 나동빈님 강의 두번째 인데 믿고 보고 있습니다! 좋은 강의 감사합니다.

할 일 (1개만 정하기, 추가 공부는 옵션)

롬복이란?

자바에서 모델 객체를 생성할 때 settet / getter / toString / hashCode / equals 메소드를 만드는데 이럴 경우 클래스 파일의 소스가 길어지고 복잡해지는데 이를 해결하기 위해 롬복을 사용한다.

롬복은 클래스 안에 필드에 대해 getter, setter의 생성이나 toString(), equals(), hashCode() 메소드, 생성자를 자동으로 생성해준다.
